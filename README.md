# quran-images

مستودع صور صفحات القرآن الكريم الجاهزة للاستخدام داخل البوت.

## الفكرة
كل صفحة محفوظة باسم ثابت داخل مجلد `pages`:

- `001.png`
- `002.png`
- `003.png`
- ...

ويتم استدعاؤها مباشرة من البوت حسب رقم الصفحة.

## هيكل المشروع

```txt
quran-images/
├─ pages/
│  ├─ 001.png
│  ├─ 002.png
│  ├─ 003.png
│  └─ ...
├─ README.md
└─ manifest.json
